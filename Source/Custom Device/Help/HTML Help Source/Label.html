Label

Identifies each ARINC 429 word. A label is encoded in the first eight bits. You can also configure hardware filtering based on the Source/Destination Identifiers (SDI) encoded in the next two bits.

This section has the following configuration settings:

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.
  • Label (decimal)—Returns the label identifier in decimal format.
  • Label (octal)—Returns the label identifier in octal format.
  • SDI Enabled—Returns whether the label will be identified and filtered based on the SDI bits.
  • SDI—Returns the SDI value determined by bits at indexes 8 and 9 in the 32-bit word.
