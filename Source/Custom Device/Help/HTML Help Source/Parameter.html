Parameter

Specifies how a value is encoded within the ARINC 429 32-bit word.

This section has the following parameters.

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.
  • Encoding Type—Returns the encoding format. Possible values are BNR, BCD, and Discrete.
  • Default Value (Scaled)—Returns the default value which is the value that will be transmitted until the parameter value is updated.
  • Start Bit—Returns the index of the first bit of the encoded parameter.
  • Number of Bits—Returns the number of bits allocated to the parameter within the 32-bit word.
  • Signed?—Returns whether the parameter value is signed which indicates whether the value range includes negative values.
  • Scale Factor—Returns the multiplier used to convert unscaled/encoded integer values to scaled/decoded floating point values.
  • Scale Offset—Returns the offset used to convert unscaled/encoded integer values to scaled/decoded floating point values.
  • Units—Returns the unit label of the scaled parameter value.
