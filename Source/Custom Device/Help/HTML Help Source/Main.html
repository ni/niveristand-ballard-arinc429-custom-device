Ballard ARINC 429

The Ballard ARINC 429 Custom Device configures Astronics Ballard hardware designed for avionics communications using the ARINC 429 protocol. This addon configures only one core of a Ballard device. You may use multiple Ballard ARINC 429 addons in your VeriStand project.

What do you want to do?

Depending on your goal, complete the following task.

Goal Task
View/Edit Hardware Configuration Configure the hardware device on this page.
Import Hardware and Parameter Configurations Import hardware and parameter configuration files in the Configuration Files section.
View Hardware Channel, Label, and Parameter Configurations Select a hardware channel, a label, or a named parameter in the system tree.

Device Configuration

This device has the following configuration settings:

  • Name—Specifies the custom device name.
  • Description—Documents this instance of the custom device.
  • Initialization Method—Specifies whether to reference the hardware device by PXI Slot Number or Card Number.
  • PXI Slot Number—Specifies the PXI Slot Number of the hardware device.
  • Card Number—Specifies the card identifier as enumerated by BTITST32 application.
  • Core Number—Specifies the hardware device core to use for communication.
