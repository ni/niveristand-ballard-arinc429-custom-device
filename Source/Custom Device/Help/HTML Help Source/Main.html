Ballard ARINC 429

The Ballard ARINC 429 Custom Device is an addon that configures Astronics Ballard hardware designed for avionics communications using the ARINC 429 protocol. This addon configures exactly one core of a Ballard device. You may use multiple Ballard ARINC 429 addons in your VeriStand project.

What do you want to do?

View / Edit Hardware Configuration

Configure the hardware device on this page.

Import Hardware and Parameter Configurations

Import hardware and parameter configuration files in the Configuration Files section.

View Hardware Channel, Label, and Parameter Configurations

View configurations by selecting a hardware channel, a label, or a named parameter in the system tree.


Device Configuration

Name: specifies the name of this custom device

Description: describes this instance of the custom device to serve as documentation

Initialization Method: specifies whether to reference the hardware device by PXI Slot Number or Card Number

PXI Slot Number: specifies the PXI Slot Number of the hardware device

Card Number: specifies the card identifier as enumerated by BTITST32 application

Core Number: specifies the hardware device core to use for communication
