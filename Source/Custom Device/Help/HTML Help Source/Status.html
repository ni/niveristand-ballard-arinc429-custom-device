Status

Counts and returns Astronics Ballard hardware errors from the configured reader in a channel.

This addon may also return timing information in status channels to indicate key operation execution times.

This section has the following configuration settings:

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.

This section contains the following channels:

  • Receiving Error Count—Count of words where an error occurred receiving the word. The specific error types are the following:
    • Bit Timing Error Count—Count of errors that occurred while decoding the bits of the word (short bits or long bits).
    • Gap Error Count—Count of when a gap of less than four bit times preceded the word.
    • Long Word Error Count—Count of words with more than 32-bits detected.
    • Parity Error Count—Count of words where a parity error was detected.
    • Timeout Error Count—Count of instances when the decoder timed out while receiving a word (short word).
