Channel

The VeriStand channel updated by this custom device.

You can perform the following actions with this channel:

  • View on a screen.
  • Assign an alias.
  • Log values.
  • Map to other channels.
  • Connect to other custom devices.

This channel has the following configuration settings:

  • Name—Specifies the name of the channel in the system definition.
  • Description—Describes the channel.
  • Default Value—Specifies the default value of the channel.
  • Channel Settings—Returns properties of the channel.
