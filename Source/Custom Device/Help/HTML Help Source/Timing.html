Timing

Contains timing channels used for understanding execution time of the custom device. These are updated each iteration of the custom device.

  • Rx Execution Time—The execution duration of the Rx execution unit in microseconds.
  • Tx Acyclic Execution Time—The execution duration of the Tx Acyclic execution unit in microseconds.
  • Tx Scheduled Execution Time—The execution duration of the Tx Scheduled execution unit in microseconds.
