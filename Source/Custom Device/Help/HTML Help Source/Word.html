Word

ARINC 429 Message

Each message includes one or more parameter values encoded into a 32-bit word as defined in the ARINC 429 protocol. A typical 32-bit word is formatted according to the table below.

Field Name Start Bit Index Field Length
(Number of Bits)
Label 0 8
SDI 8 2
Data 10 19
SSM 29 2
Parity 31 1

Note: multiple parameters may be encoded within the data field. Parameters should not overlap with each other nor with reserved bits for Label, SDI, SSM, or Parity.

Name: specifies the name of the section in the system definition

Description: describes the section
