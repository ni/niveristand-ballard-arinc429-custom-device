Word

Each ARINC 429 message includes one or more parameter values encoded into a 32-bit word as defined in the protocol.

The following table displays how to formatt a typical 32-bit word.

Field Name Start Bit Index Field Length
(Number of Bits)
Label 0 8
SDI 8 2
Data 10 19
SSM 29 2
Parity 31 1

Multiple parameters may be encoded within the data field. Parameters should not overlap with each other. They should also not overlap with reserved bits for Label, SDI, SSM, or Parity.

This section has the following configuration settings:

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.
