Ports

Hardware channels on Astronics Ballard ARINC 429 addons must be configured for either Incoming (Rx) or Outgoing (Tx).

Some hardware models support configurable direction per channel. Other models have a dedicated hardware channel range reserved for Rx and another range reserved for Tx. Refer to the hardware documentation for device capabilities.

This section has the following configuration settings:

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.
